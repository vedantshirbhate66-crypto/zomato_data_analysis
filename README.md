# 🍽️ Zomato Data Analysis Project

> Exploratory data analysis of Zomato restaurant data — uncovering dining trends, customer preferences, ratings, and online vs offline order insights using Python, Pandas, Matplotlib & Seaborn.

---

## 📌 Project Overview

This project analyzes a Zomato restaurant dataset to answer key business questions about customer behavior, restaurant types, ratings, and spending patterns.

---

## 🔍 Key Questions Explored

- 🏪 What types of restaurants are most common on Zomato?
- 🗳️ Which restaurant type receives the most votes?
- ⭐ How are restaurant ratings distributed?
- 💑 What is the average spending by couples?
- 📱 Do online orders receive better ratings than offline orders?
- 🔥 What does the heatmap reveal about online ordering by restaurant type?

---

## 📊 Key Findings

| Analysis | Conclusion |
|----------|------------|
| Restaurant Types | Majority fall in the **Dining** category |
| Votes by Type | **Dining** restaurants received the most votes |
| Ratings Distribution | Most restaurants rated between **3.5 – 4.0** |
| Couple Spending | Most couples prefer restaurants costing ~**₹300 for two** |
| Online vs Offline | **Online orders** received higher ratings than offline |
| Heatmap Insight | Dining restaurants dominate offline; cafes lean online |
